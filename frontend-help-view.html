Help & Documentation

Quick Start Guide
Getting Started
  1. Setup: Complete the initial setup by configuring admin users and connecting your Google Sheet
  2. Data Management: Navigate to the Data Management section to view, add, edit, or delete records
  3. User Management: (Admin only) Add or remove users from the Settings page
  4. Settings: Customize application preferences and manage connections
  5. Auto-refresh: Data automatically refreshes every 30 seconds by default
Pro Tip

Use keyboard shortcuts: Ctrl+R to refresh data manually.
